# Estructura Completa del Código

## Lección 1: Introducción a JavaScript

* Usar console.log() para mostrar mensajes
* Usar alert() para notificaciones del navegador
* Ejemplos de salida básica

## Lección 2: Variables, Expresiones y Condicionales

* Crear variables (let, const, var) con números, cadenas de texto y booleanos
* Todas las operaciones matemáticas (+, -, *, /, %)
* Estructuras condicionales (if, else if, else)
* Operadores relacionales (>, <, >=, <=, ===, !==)
* Operadores lógicos (&&, ||, !)
* Bucles combinados con condicionales

## Lección 3: Arreglos y Bucles

* Crear arreglos con elementos de cadena
* Arreglos anidados (matrices)
* Bucles for, while y do-while
* Bucles for-of (sintaxis moderna)
* Métodos de arreglos (push, pop, unshift, slice)
* Iterar a través de arreglos anidados

## Lección 4: Funciones

* Funciones regulares con parámetros y valores de retorno
* Funciones para cálculos (sumar, restar, multiplicar, dividir)
* Funciones que retornan objetos
* Funciones flecha (sintaxis moderna ES6)
* Usar funciones con métodos de arreglos (map, filter)
* Funciones de orden superior

## Lección 5: Objetos

* Crear objetos con propiedades y valores
* Objetos con métodos
* Usar la palabra clave 'this'
* Arreglos de objetos
* Acceder a propiedades con notación de punto y notación de corchetes
* Objetos anidados
* Estructuras de objetos complejas
